# What is Better Specs

## Summary

Use Better Specs as a concise Rails testing style guide focused on readable, behavior-driven RSpec specs.

## Rules

- Treat the guide as a set of practical best practices for Rails specs.
- Favor examples and patterns that scale to production Rails apps.
- Apply the guidance consistently across team code reviews.
